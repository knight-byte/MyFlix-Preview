MyFlix is Netflix Redesigned Clone Android App. This Repo is for progress update and App preview.

### Feature

- **Get Latest Updates on Movies**
- **Directly Play Media** without ads
- **Search** Movies/ Tv Shows
- **Clean User Interface**
